# 찾아고 🗺️

서울시 내 공공 화장실과 쓰레기통 위치를 공유하고 찾을 수 있는 모바일 앱입니다.

## ✨ 주요 기능

- 📍 서울시 내 공공 화장실, 쓰레기통 위치 확인
- ✅ 사용자 위치 기반 새 장소 추가
- 🔐 카카오톡 소셜 로그인
- 🗺️ 카카오맵 연동
- ⭐ 장소 평가 및 리뷰
- 📱 iOS & Android 지원

## 🛠️ 기술 스택

### 프론트엔드
- React Native (Expo EAS)
- 카카오맵 SDK
- 카카오 로그인
- Expo Location

### 백엔드
- Java 17 + Spring Boot 3.x
- Spring Security + JWT
- Spring Data JPA
- 카카오 OAuth 2.0

### 인프라
- AWS 컨테이너 (ECS/EKS)
- Neon DB (PostgreSQL Serverless)
- Docker
- GitHub Actions
